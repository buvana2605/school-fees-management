3. How to Run

Extract the files into your local storage. Run from index.html in local environments like VS Code.

4. Modules
i.  about.html : Gives overview of School. Contact details etc.
ii. add_student.html : Can add a student info like Name ,Class ,Grade etc.
iii.contact_us.html : can contact via email regarding feedback.
iv. index.html : Home page. Navigates to login and other pages.
v.  login_redirect.html : redirects to admin dashboard page
vi. login.html : logins through credentials. username: admin, password: admin123
vii.view_students.html : can view all the students' data.
